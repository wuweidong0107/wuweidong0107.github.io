2020-04-19 开始折腾

周末回了趟老家，只推进了一点点，惭愧惭愧。

目前做了什么？

  1. 开始阅读官方文档，初步了解Jellyfin
  2. 在PC Ubuntu-16.04 通过apt-get install 的方式安装Jellyfin：
    • 结果：失败;
    • 原因：Jellyfin已经不再对Ubuntu-16.04进行支持;
    • 解决方法：官方推荐使用docker;
  3. 在K1-Plus + OpenWrt 上使用 docker 运行 Jellyfin：
    • 结果：成功，可通过 IP 访问到 Jellyfin server;
  4. 在K1-Plus + OpenWrt 上通过 samba 挂载华硕路由器上的硬盘(我个人的简易NAS，上面存在我的各种视频文件，包括电影、美剧、家庭影片)
    • 结果：失败
    • 原因：定位中;

接下来要做什么？

  1. 了解如何使用 Docker;
  2. 解决无法通过 Samba 挂载NAS硬盘的问题;
  3. 阅读更多官方文档;
